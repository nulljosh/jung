# Jung Language

## What
Jungian programming language. Bytecode VM, C99, zero deps.
Based on jot engine.

## Build
make clean && make

## Style
- C99 strict
- snake_case everything
- No dynamic allocation outside GC
- Keep total LOC under 5000
